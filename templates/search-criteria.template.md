# Job Search Criteria

This file drives the `job-search` skill: every search run reads it top to bottom,
matches postings against it, and scores fit 1–5 explicitly against what you wrote here.
Keep it current — stale criteria produce stale shortlists.

How to fill it in:

- Replace the empty bullets under each heading with your own answers. The guidance
  comments (`<!-- ... -->`) can stay or go; the skill ignores them.
- Be specific. "Software engineer" matches everything; "Staff backend engineer,
  payments infrastructure" matches jobs you would actually take.
- A fully worked example for a fictional person is at the bottom. Delete it once
  you have filled in your own criteria.
